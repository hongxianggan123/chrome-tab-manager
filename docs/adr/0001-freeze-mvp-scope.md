# Freeze MVP Scope Around Core Tab Inventory

MVP scope is frozen around the side panel tab inventory, duplicate recognition, single-tab jump/close/archive, archive restore/delete, search, and basic status filtering. Features such as custom groups, sidebar pinning, bulk actions, duplicate prompts, dormant auto archive, memory metrics, settings, and keyboard shortcuts are intentionally deferred so the first implementation can validate the core tab and archive model without carrying the full product surface.
