# Chrome Tab Manager

一个 Chrome 扩展项目，用来管理过多标签页、减少重复标签页、归档暂时不用的页面，并在侧边栏中提供统一的标签页清单。

## 当前阶段

MVP 实现阶段已启动。当前已有 Vite + React + TypeScript + shadcn/ui + MV3 的最小扩展骨架，`dist/` 可作为 unpacked extension 加载。

## 常用命令

```bash
npm run dev
npm run test
npm run typecheck
npm run build
```
